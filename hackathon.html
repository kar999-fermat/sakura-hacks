The Hackathon Experience

A fast-paced sprint where your ideas turn into community software solutions. Pick the track that fits your skill level best.

Explorers Track

Designed for 6th-8th graders exploring code for the first time. Teams build interactive games or tools using Scratch block environments.

Pioneers Track

Designed for students with minor experience in text-based languages. Build clean web apps running Python, HTML/CSS, or Java compilers.

The Friday Timeline

03:30 PM
Arrival & Fuel Up: Check in at the Library Media Center desk, find your team table, and grab complimentary pizza slices and drinks.
03:45 PM
Opening Ceremony: Introduction of the mentor staff, rules review, and reveal of the challenge details.
04:00 PM
Hacking Sprint: Brainstorming ideas, confirming environments, and executing core software engineering.
06:15 PM
Code Freeze: Hand tools down. All project modifications stop, and teams display completed screens.
06:20 PM
Exhibition Judging: Judges circulate the space. Teams deliver live project feature demonstrations.
06:45 PM
Closing Ceremony: Distribution of rank certificates and top-tier technical gear/gift card prizes.
